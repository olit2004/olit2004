###  Featured Projects

####  [HealthNet](https://github.com/olit2004/HealthNet)
*Centralized Healthcare Management System*
- **Core**: React 19, Express 5, Prisma (PostgreSQL).
- **Features**: RBAC for Doctors/Patients, **QR Safe** emergency portal, and automated lab results.
- **Highlights**: Role-based routing and secure medical record exports.

####  [Ignite Academy](https://ignite-academy.netlify.app/)
*Advanced Learning Management System*
- **Core**: React 19, Tailwind 4, Stripe API.
- **Features**: Course marketplace, real-time progress tracking, and sales analytics for admins.
- **Highlights**: Implemented secure payments and automated email notifications.

####  [Stack Shelf](https://shelf-stack.netlify.app/demo)
*AI-Powered Technical Learning Organizer*
- **Core**: Node.js, MongoDB, Google OAuth 2.0.
- **Features**: Gmail newsletter sync, **AI labeling** for categorization, and a clean learning dashboard.
- **Highlights**: Bridges the gap between passive reading and active learning.

####  [Y-Chat](https://github.com/olit2004/Y-Chat)
*Real-time Messaging Platform*
- **Core**: MERN Stack + Socket.io.
- **Features**: Group chats, dual-theme support, and real-time "seen" status.
- **Highlights**: Seamless image sharing and message pagination.
